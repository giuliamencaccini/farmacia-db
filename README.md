# 💊 Pharmacy Database System

## 📌 Description

This project was developed for a **Database course exam**.

It represents a relational database system for managing a pharmacy, implemented using **MySQL** and a Java application developed in **IntelliJ IDEA**.

---

## ⚙️ Technologies

* Java
* MySQL
* JDBC
* IntelliJ IDEA

---

## ▶️ How to run

1. Create the database in MySQL
2. Import the schema

3. (Optional) Insert sample data

4. Configure database connection in Java:

5. Run the application from IntelliJ

---

## 🎯 Features

* Manage drugs and categories
* Manage suppliers
* Record sales
* Check product availability
